## installation

````bash
    $ git clone https://github.com/marcusasplund/stadsplanen.git

    $ cd stadsplanen

    $ yarn

    $ yarn start
````

Open up application at http://localhost:4000/ in browser

## build a release

````bash
    $ yarn build

````
This will generate a build directory with your minified/rev'd assets.
